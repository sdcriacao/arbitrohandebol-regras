# 🤾 Regras de Jogo — Handebol Indoor

> Aplicativo interativo de estudo para árbitros de handebol, baseado no **Catálogo Oficial de Perguntas e Respostas da IHF** — Edição Maio de 2024.

---

## 📖 Sobre o Projeto

Este app foi desenvolvido para facilitar o estudo e a preparação de árbitros de handebol, oferecendo uma experiência de aprendizado completa, interativa e diretamente vinculada às **Regras de Jogo Handebol Indoor — Edição 1º de Março de 2025** da Confederação Brasileira de Handebol (CBHb).

A navegação é feita por meio de uma **quadra de handebol interativa**: cada zona da quadra representa uma Regra. Basta tocar para iniciar os exercícios.

---

## ✨ Funcionalidades

- 🏟️ **Quadra interativa** — navegação visual por todas as 18 Regras + Regulamento da Área de Substituição
- 📋 **422 questões** baseadas no catálogo oficial da IHF, em Português do Brasil
- ✅ **Múltipla seleção** de alternativas — acerto parcial conta como erro
- 🎵 **Sons de apito** temáticos — 2 silvos para acerto (Gol!) e 1 silvo longo para erro (Falta!)
- 📖 **Catálogo de Respostas** vinculado a cada questão
- 📜 **Regra de Jogo** com o artigo específico da resposta, incluindo sub-alíneas
- ⏱️ **Temporizador** por Regra — conta o tempo de estudo, zera ao retornar
- 🎨 **Estados visuais** na quadra: laranja (disponível), azul (em andamento), verde (concluída)
- 🔙 **Navegação por histórico** — botão Retornar em todas as telas
- 📊 **Resultado final** por Regra com percentual de acertos
- 📱 **Responsivo** — funciona no computador e no celular pelo navegador

## 🚀 Como Usar

### ▶️ Opção 1 — Direto no navegador (sem instalação)

Acesse o link do GitHub Pages:

```
https://sdinformatica.github.io/handebol
```

### 💾 Opção 2 — Rodar localmente

1. Faça o download do arquivo `index.html`
2. Abra diretamente no navegador (Chrome, Firefox, Edge, Safari)
3. Nenhuma instalação ou conexão com internet é necessária

> ✅ O app é um **único arquivo HTML autocontido** — funciona 100% offline após o carregamento.
